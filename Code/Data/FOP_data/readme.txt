This folder contains the FOP data for the province of Québec, Canada, shared by the Canadian Forest Service.

The underlying models are described in:
Nadeem, K., Taylor, S.W., Woolford, D.G. and Dean, C.B., 2020. Mesoscale spatiotemporal predictive models of daily human-and lightning-caused wildland fire occurrence in British Columbia. International journal of wildland fire, 29(1), pp.11-27.
